苹果笔记本MacbookPro重置密码的方法

前几天，恰好有位老领导委托他人拿来了一台苹果笔记本，好像是 Macbook Pro 2018 吧，由于主机安装时候使用上的需要，安装了 Windows 系统，老是提示 C 盘容量不足报警。因此委托我进行容量调整。

距离当年在电脑城体验的苹果主机，转眼间已有了 10 年，领导说，如果可以安装双系统到时挺好的，我随手摘网上找了一下，果真是有一些资料，可是都比较零散，因此这里总结一下；也许对个别人来说，也许用得上。

配置 U盘启动或者切换系统的方法

经过检查，发现 Windows 系统中已经存在 Boot Camp 软件，输入“恢复”后找到相应的菜单即可进入。不过查了一下，Macbook 没有 BISO 等设置，所谓的进入 U盘 PE 或者苹果系统的方法是：按住 Option 键，然后按电源键开机，那么就会出现一个启动项界面，从而实现改变启动方式的功能。

![Macbook启动项界面-HHJ](static/img/Macbook-开机启动项界面-20200824.jpg)

有意思的是，进入该界面后，再接入 U盘 PE，那么也能很快识别出来；跟我之前在 BISO 里边配置 U盘启动还是有明显差别的。

Macbook 重置密码的方法

恰好领导也并不知道自己 MacOS 的系统开机密码，自己结合网络资料测试了一下，操作流程如下：

  1. 按住 Option 键，然后按电源键开机；

  2. 按方向键选择 Macintosh （苹果） 系统，按 Command + r 后自动进入系统恢复界面；

  3. 在经历进度条界面一会后，选择界面的操作语言后，出现以下界面，执行菜单命令 使用工具→终端 ，弹出终端界面：

  4. 在终端中执行 resetpassword ，在弹窗中选择需要重置的用户名，并填入新密码，然后存储到系统即可完成修改。

  5. 完成修改后重新使用第 1 步的方法进入苹果系统，在登入后可能会提示钥匙串密码是否修改等内容，我选择了修改成一致后，重复登录检查账户使用正常。

需要注意的是，上述方法应当使用在合法授权的电脑上，请勿滥用！践行社会主义核心价值观，人人有责！
